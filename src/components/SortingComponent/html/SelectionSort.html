The selection sort algorithm sorts an array by repeatedly finding the minimum element (Consider ascending order) from unsorted part

Drawbacks:

The primary disadvantage of the selection sort is its poor efficiency when dealing with a huge list of items. Similar to the bubble sort, the selection sort requires n-squared number of steps for sorting n elements.

Algorithm how it works with example:

arr = [40, 50, 20, 30, 10]

Pick the first element from the list 40 Compare this value against the rest of the item [1..4] in the list to get the minimum value from the list

minimum item from the list [2..4] is 10, now swap 40 and 10 poistion. Result Array will be as below

arr = [10, 50, 20, 30, 40]

Now the pointer is moved to second item which is 50. Compare this value against the rest of the item [2..4] in the list to get the minimum value from the list

minimum item from the list [2..4] is 20, now swap 50 and 20 poistion. Result Array will be as below

arr = [10, 20, 50, 30, 40]

Now the pointer is moved to next item which is again 50. Compare this value against the rest of the item [3..4] in the list to get the minimum value from the list

minimum item from the list [3..4] is 30, now swap 50 and 30 poistion. Result Array will be as below

arr = [10, 20, 30, 50, 40]

Now the pointer is moved to next item which is again 50. Compare this value against the rest of the item which is the last item in the list to get the minimum value from the list

minimum item is 40, now swap 50 and 40 poistion. Result Array will be as below

arr = [10, 20, 30, 40, 50]

Javascript Logic 

  function selectionSort(arr) {
   var n = arr.length;
   for (let i = 0; i < n - 1; i++) {
    var min_idx=i;
    for (var j=i+1; j < n; j++) if (arr[j] < arr[min_idx]) min_idx=j; 
    var temp=arr[min_idx]; 
    arr[min_idx]=arr[i]; 
    arr[i]=temp; 
  }
