Insertion sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time.

Drawback:

It is much less efficient on large lists than more advanced algorithms

Algorithm how it works with example:

arr = [40, 50, 20, 30, 10]

First pick the first element and compare against the next of the item.

In our case compare 40 with 50, to see if first element is greater than next element. If yes pick the smallest element and insert in the first spot.

If the first element is less than the second element, we can move to next element. Now the pointer moves to second element. Which is 50.

Now compare this with the next element which is 20. Since 50 > 20, we have to pick 20 and start checking backwards in the list to see where to insert this number. Once we indentify the spot, insert 20 in that spot.

In our case, the spot is 0. The update list after the above process is ,

arr = [20, 40, 50, 30, 10]

Next pointer moves to poistion 3, value is 50 compare against 30. Since 50 > 30, insertion process is performed again. Now the resul will be,

arr = [20, 30 40, 50, 10]

Same needs to be performed for the next two items [50 & 10]

arr = [10, 20, 30 40, 50] 

 function insertionSort(arr) {
  var n = arr.length;
  var key, j;
  for (var i = 1; i < n; ++i) {
   key = arr[i];
   j = i - 1;

   while (j >= 0 && arr[j] > key) {
    arr[j + 1] = arr[j];
    j = j - 1;
   }
   arr[j + 1] = key;

  }
 }
